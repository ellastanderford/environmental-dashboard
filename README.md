# Environmental Climate Dashboard

A Python-based data exploration project designed to analyze and visualize global climate trends.

## 📌 Project Overview
This dashboard extracts insights from climate datasets to showcase changes over time. The goal is to track specific environmental metrics (like temperature anomalies or CO2 levels) and display them in a clean, readable graph.

## 📊 The Dataset
* **Source:** Kaggle
* **Dataset Name:** Air Quality UCI Dataset
* **Key Variables tracked:** Date, Temperature (T)
